Terraform

    What is Terraform?

  • Terraform is an infrastructure-as-code software tool created by HashiCorp. Users define and provide data center infrastructure using a declarative configuration language known as HashiCorp Configuration Language (HCL) or optionally as JSON. Terraform makes managing and orchestrating large, multi-cloud infrastructures easier. .

    • Features

  • Infrastructure as code

  • Multi-cloud provisioning

  • Kubernetes management

  • Network infrastructure management

  • Manage virtual machines

  • Integrate with existing workflows.

  • Enforce policy as code

  • Inject secrets into Terraform
