Trello

What is Trello

Trello is a web-based application for creating lists in a kanban-style, developed by Atlassian's subsidiary Trello Enterprise. Originally created by Fog Creek Software in 2011, it was spun off as a separate company based in New York in 2014 and was acquired by Atlassian in January 2017.

Features

Users can create their own task boards with different columns and move tasks between them. Columns typically represent task statuses like To Do, Progress, Done. The tool can be used for personal and business purposes, including real estate management, software project management, school bulletin boards, scheduling classes, accounting, web design, playing games, and managing cases for law firms.
