Star UML

What is StarUML?

StarUML is a software engineering tool for system modeling using a unified modeling language, as well as system modeling language and classical modeling symbols. It is released by MKLab and is available on operating systems Windows, Linux, and MacOS.

Features

  • 1. Composite Structure Diagrams
  • 2. Component Diagrams
  • 3. Object Diagrams
  • 4. Package Diagrams
  • 5. Use Case Diagrams
  • 6. Activity Diagrams
  • 7. Sequence Diagrams
  • 8. Communication Diagrams
  • 9. Timing Diagrams
  • 10. State Machine Diagrams
  • 11. Data Flow Diagrams
  • 12. Interaction Overview Diagrams
  • 13. Profile Diagrams
